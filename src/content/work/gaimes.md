Web3 game publishing and discovery platform where players find and play games, and developers publish, monetize, and manage their projects on-chain. Designed the full product experience — from game discovery and detail pages to developer profiles, a 5-step game submission wizard, and a token-based monetization system.

## Scope

- Designed the homepage discovery experience with a featured game hero, personalized "You're Playing" feed, and game card grid with free/premium and play count indicators
- Built the game detail page with media gallery, game stats, creator info, latest commits, and comments section — bridging game discovery with developer transparency
- Designed developer profile pages showing games created, games played, deployment status, and creator stats
- Created a 5-step game submission wizard: game information, environment setup (frontend/backend config), earning method selection, repository linking, and final submission
- Designed the monetization layer — Free to Play vs. Premium Game model with GAIMES token pricing, platform fee breakdown, and creator earnings
- Defined a dark design system with pink accent and playful typography consistent with the gaming platform identity

## Impact

- Unified game discovery and developer tooling into a single cohesive platform experience
- Token-based monetization model gives developers a direct earn path from day one with transparent fee structure
- 5-step submission wizard reduced game publishing complexity for non-technical game creators
