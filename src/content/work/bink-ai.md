Web3-native AI assistant combining the familiar chat interface of modern LLMs with deep crypto tooling — wallet connection, multi-chain network management, and conversations tailored for DeFi, trading, and on-chain tasks. Designed the full product experience from onboarding and wallet connect to the core chat interface, network panel, and search.

## Scope

- Designed the wallet connect onboarding flow — clean entry screen supporting Phantom, Binance Web3 Wallet, Trust, and Safepal with a minimal, non-intimidating UI for crypto newcomers
- Built the core chat interface with persistent conversation history (grouped by recency), a "Ask anything" input, and context-aware responses tailored to Web3 topics
- Designed the Network Connect side panel for managing multi-chain connections — supporting Base, Polygon, BSC, TronNRG, Solana, Ethereum, Monad, Avalanche, and Near with one-tap connect and bulk disconnect
- Created the chat search overlay for fast navigation across conversation history
- Defined a dual-mode design system — light onboarding, dark chat interface — balancing approachability with the crypto-native aesthetic users expect
- Designed conversation UI patterns for structured AI responses including formatted explanations, inline suggestions, and follow-up prompts

## Impact

- Bridged the UX gap between mainstream AI assistants and Web3 tooling — users get a familiar chat experience without sacrificing on-chain depth
- Multi-chain network management consolidated into a single panel, eliminating the need to switch between wallets and dApps for network switching
- Wallet-first onboarding establishes identity and context from session start, enabling personalized, wallet-aware AI responses
